## 支持的Markdown语法
- 但是支持大部分其它功能
- 插图也是支持的
- "\-" 例如这个是支持的
- 支持代码高亮
 - 如：
 ```
 int main（）
 {
    return 0;
 }
 ```
- 支持链接

- 支持斜体加粗
 - 如：*Example*
- 支持文本高亮
 - `如`
- 支持引用
 
 >如这样
  
## 不支持的Markdown语法
- 不支持分割线
```
    =========分割线好像并不支持
```
- github的.md不支持LaTex
- 不支持流程图什么的
- 不支持toc
